# Theme Rename Plan — IP Compliance

## Principle
Names must be original and not use trademarked terms.
Descriptions can say "inspired by" a *genre* or *aesthetic era*, never a specific franchise.
Fan recognition comes from the visual aesthetic + clever naming, not direct references.

## Notes
- "Beskar Forge" might still be too close (beskar is SW-specific). Alternatives: **Forged Steel**, **Worn Alloy**, **Ironclad**
- "Vault Terminal" could evoke Fallout too directly. Alternatives: **Bunker Terminal**, **Shelter CRT**
- "Omni Flare" might reference omni-tool. Alternatives: **Holo Gauntlet**, **Warm Holo**, **Solar Flare**
- "Blue Cortex" could evoke Cortana. Alternatives: **Blue Visor**, **Combat Holo**, **Tactical Blue**
- Group names "Galactic" and "Federation" are generic enough to be safe
- All descriptions should be rewritten to describe the *aesthetic* not the *source*
